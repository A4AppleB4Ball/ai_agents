# Google Search — Page Map

## Pages

### Homepage
- **URL:** `https://www.google.com`
- **Purpose:** Search entry point
- **Key elements:** Search textarea, Search button, I'm Feeling Lucky button, Google logo
- **Navigation:** Submit search → Results page

### Search Results
- **URL:** `https://www.google.com/search?q={query}`
- **Purpose:** Display search results for a query
- **Key elements:** Results list, result headings, snippets, pagination, search tools
- **Navigation:** Click result → External page; Click "Next" → Next results page; Modify search → Updated results

### Image Results
- **URL:** `https://www.google.com/search?q={query}&tbm=isch`
- **Purpose:** Display image search results
- **Key elements:** Image grid, image preview overlay
- **Navigation:** Click image → Preview overlay; Click "All" tab → Regular results

## Navigation Flow

```
Homepage → [submit search] → Results Page
Results Page → [click result] → External Page
Results Page → [click "Images"] → Image Results
Results Page → [click "Next"] → Results Page (page 2)
External Page → [browser back] → Results Page
```
